계정 삭제 안내

MyFoodLoad 계정을 삭제하면 다음 데이터가 영구적으로 삭제되며 복구할 수 없습니다.

삭제되는 데이터

  • 계정 정보 (이메일, 이름, 프로필 사진)
  • 유튜브 좋아요 분석 결과 및 음식 취향 데이터
  • 즐겨찾기한 맛집 목록
  • 방문 완료 기록
  • 맞춤 추천 이력

삭제 방법

  1. MyFoodLoad 앱을 실행합니다.
  2. 프로필 탭 → 설정 (톱니바퀴 아이콘)을 누릅니다.
  3. 계정 삭제 버튼을 누릅니다.
  4. 확인 대화상자에서 삭제를 누르면 즉시 삭제됩니다.

문의: myfoodload.support@example.com
