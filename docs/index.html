MyFoodLoad

YouTube 좋아요 영상을 분석하여 나만의 맛집을 추천해 드립니다.

서비스 소개

MyFoodLoad는 사용자의 YouTube '좋아요' 영상 메타데이터를 분석하여 음식 취향을 파악하고, 현재 위치를 기반으로 개인화된 맛집을 추천하는 서비스입니다.

주요 기능

  • YouTube '좋아요' 영상 분석을 통한 음식 취향 파악
  • 현재 위치 기반 개인 맛집 추천
  • 인기 맛집(트렌딩) 추천
  • 카카오맵 기반 지도 표시 및 길안내
  • 맛집 즐겨찾기 및 방문 기록 관리
